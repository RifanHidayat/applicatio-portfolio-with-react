# Dokumentasi Ringkas Proyek React/Next.js

Kumpulan proyek React/Next.js yang pernah saya kerjakan, lengkap dengan deskripsi singkat dan tautan pratinjau.

   Portofolio modern dengan fokus pada performa dan aksesibilitas.  
   Preview: https://portofolio-2-nine-black.vercel.app/  
   Rujukan konten: [Rifinity Portfolio](https://portofolio-2-nine-black.vercel.app/)

2. **Portfolio v1 – Next.js + TypeScript**  
   Versi sebelumnya dengan struktur sederhana dan fokus pada profil pribadi.

3. **Company Profile – Next.js + TypeScript**

4. **Simple Project – React + Vite + Zustand**  
   Contoh state management ringan menggunakan Zustand.  
   Preview: https://react-view-zustand-simple-project.vercel.app/

5. **Portfolio (Personal) – Next.js + TypeScript**  
   Portofolio pribadi dengan komponen reusable dan gaya konsisten. Preview sesuai tautan yang diberikan di atas.

## Teknologi Utama

- Next.js, React, TypeScript
- Vite (untuk proyek Zustand)
- Tailwind CSS (umum pada proyek UI)
- Zustand untuk state management ringan
